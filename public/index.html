Users API

Local development endpoints for http://localhost:3000

PUT http://localhost:3000/users/feed

Retrieve a compatibility-sorted feed of recommended users for a given Spotify user.

Method: PUT

Endpoint:/feed

Request

{
  "spotifyId": "user123",
  "genres": ["pop", "rock"]
}

Success Response :

(200 OK) 
[
  {
    "_id": "67301c72ab12df001fe3ab88",
    "spotifyId": "otherUser1",
    "displayName": "Alice",
    "profileImage": "https://image.jpg",
    "profileSongs": [ ... ],
    "genres": [ "pop", "rnb" ],
    "matches": [],
    "liked": [],
    "passed": [],
    "isBot": false,
    "compatibility": 0.72
  }
]

PATCH http://localhost:3000/users/matches

Register user's like/disliked

Method: PATCH

Endpoint:/users/matches

Request

{
  "spotifyId": "user123",
  "otherSpotifyId": "user987",
  "isLike": true
}

Success Response :

(200 OK) 
{
    "spotifyId": "matchedUser1",
    "displayName": "Alice",
    "profileImage": "...",
    "genres": [...],
    "matches": [...],
    "liked": [...],
    "passed": [...],
    "profileSongs": [ ... ],
    "isBot": false
  }

POST http://localhost:3000/users

Update an existing user and add new user

Method: POST

Endpoint:/users

Request

{
  "spotifyId": "user123",
  "displayName": "John",
  "email": "john@gmail.com",
  "profileImage": "https://image.jpg",
  "genres": ["pop", "rock"]
}

Success Response :

(200 OK) 
{
    "spotifyId": "matchedUser1",
    "displayName": "Alice",
    "profileImage": "...",
    "genres": [...],
    "matches": [...],
    "liked": [...],
    "passed": [...],
    "profileSongs": [ ... ],
    "isBot": false
  }
