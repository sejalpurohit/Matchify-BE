Users API

Local development endpoints for http://localhost:3000

GET http://localhost:3000/users

Retrieves a list of all user records stored in the database

Method: GET

Endpoint:/users

Success Response :

(200 OK) 
[
  {
    "_id": "67301c72ab12df001fe3ab55",
    "name": "John Doe",
    "email": "john@example.com",
    "age": 28
  },
  {
    "_id": "67301c72ab12df001fe3ab56",
    "name": "Jane Smith",
    "email": "jane@example.com",
    "age": 31
  }
]

PUT http://localhost:3000/users/feed

Add user details with top 5 song from user's spotify feed

Method: PUT

Endpoint:/feed

Request

{
  "spotifyId": "user123",
  "genres": ["pop", "rock"]
}

Success Response :

(200 OK) 
[
  {
    "_id": "67301c72ab12df001fe3ab88",
    "spotifyId": "otherUser1",
    "displayName": "Alice",
    "profileImage": "https://image.jpg",
    "profileSongs": [ ... ],
    "genres": [ "pop", "rnb" ],
    "matches": [],
    "liked": [],
    "passed": [],
    "isBot": false,
    "compatibility": 0.72
  }
]

PATCH http://localhost:3000/users/matches

Register a like/disliked users and update matches

Method: PATCH

Endpoint:/users/matches

Request

{
  "spotifyId": "user123",
  "otherSpotifyId": "user987",
  "isLike": true
}

Success Response :

(200 OK) 
{
  "spotifyId": "user123",
  "matches": ["user987"],
  "liked": [],
  "passed": []
}

PUT http://localhost:3000/users/matches

Return full profiles of matched users.

Method: PUT

Endpoint:/users/matches

Request

{
  "spotifyId": "user123"
}

Success Response :

(200 OK) 
[
  {
    "spotifyId": "matchedUser1",
    "displayName": "Alice",
    "profileImage": "...",
    "genres": [...],
    "matches": [...],
    "liked": [...],
    "passed": [...],
    "profileSongs": [ ... ],
    "isBot": false
  }
]

POST http://localhost:3000/users

Update an existing user

Method: POST

Endpoint:/users

Request

{
  "spotifyId": "user123",
  "displayName": "John",
  "email": "john@gmail.com",
  "profileImage": "https://image.jpg",
  "genres": ["pop", "rock"]
}

Success Response :

(200 OK) 
{
  "spotifyId": "user123",
  "displayName": "John",
  "email": "john@gmail.com",
  "profileImage": "https://image.jpg",
  "genres": ["pop", "rock"]
}

PATCH /users

Update genres and profile songs.

Method: PATCH

Endpoint:/users

Request

{
  "spotifyId": "user123",
  "genres": ["pop", "rnb"],
  "profileSongs": [...]
}

Success Response :

(200 OK) 
{
  "spotifyId": "user123",
  "genres": ["pop", "rnb"],
  "profileSongs": [...]
}
